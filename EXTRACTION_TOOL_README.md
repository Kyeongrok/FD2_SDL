# FD2游戏资源提取工具使用指南

## 概述
基于IDA Pro MCP服务器对fd2.exe的深度逆向工程分析，我们开发了FD2游戏资源提取工具。该工具能够解析和提取ANI.DAT动画资源文件和FDOTHER.DAT图形资源文件。

## 文件说明

### 主要工具
- **fd2_extractor.py**: 主要的资源提取工具
- **extracted_resources/**: 提取的资源和分析报告目录
- **EXTRACTION_TOOL_README.md**: 本使用指南

### 生成的分析报告
1. **ani_dat_analysis.json**: ANI.DAT文件结构分析报告
2. **fdother_dat_analysis.json**: FDOTHER.DAT文件结构分析报告
3. **extract_all_resources.py**: 自动生成的完整资源提取脚本

## 使用方法

### 1. 基本提取
```bash
# 提取当前目录下的所有DAT文件
python fd2_extractor.py

# 指定工作目录
python fd2_extractor.py /path/to/game/files
```

### 2. 查看分析结果
提取完成后，在`extracted_resources`目录中：
- 查看JSON分析报告了解文件结构
- 检查提取的原始数据文件
- 使用生成的Python脚本进行批量提取

### 3. 进一步分析
基于提取的数据，可以：
1. 开发图形查看器显示提取的图像
2. 解析ANI.DAT命令系统以重现动画
3. 提取所有游戏资源（声音、文本、关卡数据等）

## 技术细节

### ANI.DAT文件格式
基于IDA Pro MCP服务器分析：
- **文件头**: "LLLLLL" (6字节)
- **资源表**: 从偏移6开始，每个条目8字节（起始偏移量+结束偏移量）
- **资源结构**:
  - 8字节头（可能包含冗余偏移量）
  - 173字节资源头
  - 块数据（每个块8字节头+可变长度数据）

### 命令系统
ANI.DAT使用基于命令的动画脚本系统：
- 每个块包含一个命令标识符（0-255）
- 命令处理函数数组`funcs_37012`（位于fd2.exe地址0x5276a）
- 已识别的命令：
  - 命令0: 填充调色板/内存区域
  - 命令1: 复制内存块
  - 命令2: 数据处理（待进一步分析）

### FDOTHER.DAT文件格式
- **文件头**: "LLLLLL" (6字节)
- **资源表**: 从偏移6开始，每个条目8字节（起始偏移量+结束偏移量）
- **资源数据**: 原始8位调色板索引图形数据
- **常见尺寸**: 320x200, 398x25, 32x321, 256x256等

## 已知限制

1. **命令解码**: 只识别了部分命令，完整的256个命令需要进一步分析
2. **图像渲染**: 提取的图形数据需要调色板信息才能正确显示
3. **资源依赖**: 某些资源可能依赖于其他资源（如调色板）

## 故障排除

### 常见问题
1. **找不到文件**: 确保ANI.DAT和FDOTHER.DAT在工作目录中
2. **内存不足**: 处理大文件时可能需要增加Python内存限制
3. **编码问题**: 报告中的十六进制数据可能需要进一步解析
