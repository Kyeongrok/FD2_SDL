## 2. 游戏主循环结构

### 2.1 main函数 (0x25E3A)

```c
while (1) {
    sub_25977(18, 0);      // 初始化游戏状态/播放音乐
    v10 = sub_25EBB();     // 检查存档/进入游戏
    
    if (v10 == 0) {
        // 继续游戏循环
        do {
            v10 = sub_117E7();  // 处理输入/状态机
            
            if (n2_0 == 1) {
                // 进入新状态
                sub_22E5C();
            }
            else if (n2_0 == 2) {
                // 执行状态处理函数
                funcs_25E23[n17]();  // 状态处理函数表
            }
        } while (!v10);
    }
    
    if (v10) {
        // 退出游戏
        sub_37ED8();
    }
}
```

## 4. 游戏状态机

游戏使用 `n17` 变量存储当前游戏状态，`funcs_25E23` 和 `funcs_25E3A` 是函数表。

```
状态 0:  标题/启动
状态 1:  开场动画
状态 2:  菜单
状态 3:  关卡选择
状态 4:  战斗/游戏中
状态 5:  暂停
状态 6:  胜利画面
```
