# FD2.EXE 启动流程完整分析

## 1. 入口点与程序架构

### 入口点
- **地址**: 0x3CCB4 (start)
- **类型**: 库函数，未反编译
- **调用**: 由 DOS/4G 运行时环境调用，跳转到 main()

### main() 函数
- **地址**: 0x25BF4
- **调用者**: 0x4609B (C运行时启动代码)

## 2. 初始化阶段函数调用顺序

### 2.1 栈/内存初始化

```
main()
  └─ sub_3702F(28)                    # 栈初始化/安全检查
```

### 2.2 音频系统初始化 (Miles Audio Interface Library)
```
main()
  ├─ sub_37D3E()                      # AIL_startup() - 音频库启动
  │    └─ fprintf(env, "AIL_startup()\n")  # 调试输出
  │
  ├─ sub_3AA72()                      # 音频驱动初始化
  │    ├─ sub_37C9C()                 # 检查音频环境
  │    ├─ sub_3F22A()                 # 环境检查
  │    ├─ sub_44240()                 # 获取音频设置
  │    └─ AIL_install_MDI_INI()       # 安装MDI驱动
  │
  ├─ sub_3908B()                      # 初始化音频驱动
  │    └─ 返回: dword_53EDC (驱动句柄)
  │
  ├─ sub_392D0(dword_53EDC)          # 分配音频句柄
  │    ├─ 返回: dword_53EE4 (第一个句柄)
  │    └─ 返回: dword_53EE8 (第二个句柄)
  │
  └─ sub_3ACA3(dword_53ED8)          # 音频配置
```

## 3. 主游戏循环与动画播放

### 3.1 主循环结构
```
main()
  └─ while(1)                         # 主循环
       ├─ sub_25977(18, 0)           # 初始化游戏状态/播放音乐
       ├─ sub_25EBB()                 # 检查存档/进入游戏状态
       │    ├─ sub_1F894()           # 播放开场动画并显示菜单
       │    ├─ sub_10010()           # 加载存档数据
       │    └─ sub_25977()           # 重新初始化
       └─ sub_117E7()                # 状态机/菜单处理
```

### 3.2 开场动画播放 (sub_1F894 - 0x1F894)

#### 加载的资源:
```
sub_1F894()
  ├─ FDOTHER.DAT (资源77)            # 加载动画数据
  ├─ FDOTHER.DAT (资源76)            # 加载背景图像
  ├─ FDOTHER.DAT (资源74)            # 加载标题图像
  ├─ FDOTHER.DAT (资源76)            # 重新加载背景
  ├─ FDOTHER.DAT (资源101)           # 加载菜单背景
  └─ FDOTHER.DAT (资源69-73)         # 加载5帧动画
```

#### 动画播放流程:
1. **初始化屏幕**
   - `memset(655360, 0, 64000)` - 清空视频内存
   - `sub_11D40()` - 设置调色板

2. **标题显示**
   - `sub_4E98D()` - 解压BG图像到视频缓冲区
   - `sub_1F525()` - 复制到屏幕
   - `sub_17AA9()` - 视频效果处理

3. **滚动动画播放**
   - 分配230,400字节缓冲区用于动画帧
   - 加载5帧动画 (FDOTHER.DAT 资源69-73)
   - 主循环 (535次迭代):
     ```
     for (n535 = 535; n535 >= 0; n535--)
     {
       sub_11EB0()           # 帧复制到视频缓冲区
       sub_1F525()           # 显示到屏幕
       
       # 特定帧处插入效果
       if (n535 == 450) sub_1F73F()  # 特效
       if (n535 == 330) sub_1F81E()  # 音效触发
       if (n535 == 210) sub_1F81E()
       if (n535 == 110) sub_1F81E()
       if (n535 == 10)  sub_1F73F()
       
       sub_10620()           # 检查按键 (可跳过动画)
       j___delay(30)         # 帧延迟
     }
     ```

4. **淡出效果**
   - `sub_2DF01()` - 调色板淡出 (40步)
   - 释放动画缓冲区

5. **菜单显示**
   - 加载 FDOTHER.DAT (资源7, 8) - 菜单数据
   - `sub_20421(1, 15, 1)` - 显示菜单文字
   - `sub_25B45()` - 显示菜单选项
   - `sub_11DF2()` - 设置调色板

6. **菜单循环**
   ```
   while (!n12)
   {
     sub_1FF79()              # 绘制菜单选项
     int386(22, &n3, &n3)     # 获取键盘输入
     
     if (key == UP) n3--;     # 上移选项
     if (key == DOWN) n3++;   # 下移选项
     if (key == ENTER/SPACE) n12 = 1;  # 确认选择
   }
   ```

7. **菜单选择确认**
   - 闪烁效果 (4次)
   - `sub_1F882()` - 清理
   - `memset(655360, 0, 64000)` - 清屏
   - 返回选择结果

## 8. 备注

1. **视频内存布局**: 0xA0000 (655360) 是 VGA 模式13h的视频缓冲区起始地址
2. **DAT文件格式**: 所有DAT文件使用统一的索引格式: 6字节头 + 4字节/资源的索引表
3. **调色板**: 使用6位调色板值，需要左移2位转换为8位
4. **内存管理**: DOS模式下使用段地址寻址，所有指针为32位线性地址
5. **中断调用**: 使用 int386() 进行DOS系统调用获取键盘输入
