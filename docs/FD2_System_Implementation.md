## 1. 구현 완료된 시스템

### 1.1 유닛 시스템 (fd2_unit.h/c)

| 모듈 | 기능 |
|------|------|
| 데이터 구조 | 80바이트 Unit 구조체, 80바이트 UnitRaw 구조체 |
| 직업 시스템 | 15종 직업, 기본 속성 + 성장치 |
| 유닛 관리 | 배열 연산, 위치 검색, 생존 체크 |
| 전투 계산 | 물리 피해, 마법 피해 |

### 1.2 세이브 시스템 (fd2_save.h/c)

| 모듈 | 기능 |
|------|------|
| 세이브 로드 | 22987바이트 파일 읽기 |
| 체크섬 검증 | sub_4DF28 + sub_4DF09 |
| 유닛 데이터 파싱 | 오프셋 4771, 유닛당 80바이트 |

## 2. 게임 코어 루프 (IDA 분석)

```
main (0x25E3A)
├── sub_25977(18, 0)          # 음악/상태 초기화
├── sub_25EBB()                # 세이브 체크
│   └── sub_1F894()           # 오프닝 애니메이션 + 메뉴
│
└── 게임 루프 (while 1)
    ├── sub_117E7()            # 입력 처리 / 상태 머신
    │   ├── 키 1/44/76        # 유닛 선택
    │   ├── 키 57/28          # 커서 이동 / 유닛 이동
    │   ├── 키 H              # 위
    │   ├── 키 P              # 아래
    │   ├── 키 K              # 왼쪽
    │   ├── 키 M              # 오른쪽
    │   └── 그 외 키          # 기타 동작
    │
    └── 상태 처리
        ├── funcs_25E23[n17]() # 상태 처리 함수
        └── funcs_25E3A[n17]() # 상태 렌더링 함수
```

```
오프셋  크기  필드              설명
+0    2    hp                현재 HP
+2    2    max_hp            최대 HP
+4    2    mp                현재 MP
+6    2    max_mp            최대 MP
+8    2    attack            공격력
+10   2    defense           방어력
+12   2    magic             마법력
+14   2    resistance        마법 저항
+16   2    agility           민첩
+18   2    luck              행운
+20   2    move_range        이동 범위
+22   2    attack_range      공격 범위
+24   2    experience        경험치
+26   2    level             레벨
+28   2    padding           정렬
+30   1    direction         방향 (0-3)
+31   1    status_flags      상태 플래그
+32   1    team              팀 (1 = 플레이어)
+33   1    ai_type           AI 타입 (2 = 적)
+34   1    padding
+35   1    padding
+36   1    unknown
+37   1    unknown
+38   1    unknown
+39   1    unknown
+40-79 40  unknown           전투 관련 데이터
```

## 4. 전투 이동 흐름 (sub_18890)

```
sub_18890(unit_index)
├── sub_1F183()               # 이동 가능 여부 체크
├── sub_4E8A5()               # 이동 타입 획득
├── sub_145CD()              # 이동 초기화
├── sub_4E390()              # 이동 범위 계산
├── sub_18B84()              # 이동 경로 계산
├── sub_115B6()              # A* 패스파인딩
├── sub_4E4F6()              # 경로 그리기
├── sub_18D8C()              # 이동 실행
│   ├── sub_177FC()          # 경로 순회 체크
│   ├── sub_1CFF0()          # 이동 애니메이션
│   └── sub_1BBDC()          # 공격 애니메이션
└── 유닛 위치 + 상태 갱신
```
