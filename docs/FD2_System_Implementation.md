# FD2 游戏系统实现报告

> 基于IDA Pro MCP服务器逆向工程分析

---

## 1. 已实现的系统

### 1.1 单位系统 (fd2_unit.h/c)

| 模块 | 功能 |
|------|------|
| 数据结构 | 80字节Unit结构，80字节UnitRaw结构 |
| 职业系统 | 15种职业，基础属性+成长值 |
| 单位管理 | 数组操作，位置查找，存活检查 |
| 战斗计算 | 物理伤害，魔法伤害 |

### 1.2 存档系统 (fd2_save.h/c)

| 模块 | 功能 |
|------|------|
| 存档加载 | 22987字节文件读取 |
| 校验和验证 | sub_4DF28 + sub_4DF09 |
| 单位数据解析 | 偏移4771处，80字节/单位 |

## 2. 游戏核心循环 (IDA分析)

```
main (0x25E3A)
├── sub_25977(18, 0)          # 初始化音乐/状态
├── sub_25EBB()                # 检查存档
│   └── sub_1F894()           # 开场动画+菜单
│
└── 游戏循环 (while 1)
    ├── sub_117E7()            # 输入处理/状态机
    │   ├── 按键1/44/76       # 选择单位
    │   ├── 按键57/28         # 移动光标/移动单位
    │   ├── 按键H             # 向上
    │   ├── 按键P             # 向下
    │   ├── 按键K             # 向左
    │   ├── 按键M             # 向右
    │   └── 其他按键          # 各种操作
    │
    └── 状态处理
        ├── funcs_25E23[n17]() # 状态处理函数
        └── funcs_25E3A[n17]() # 状态渲染函数
```

## 4. 战斗移动流程 (sub_18890)

```
sub_18890(unit_index)
├── sub_1F183()               # 检查是否可以移动
├── sub_4E8A5()               # 获取移动类型
├── sub_145CD()              # 初始化移动
├── sub_4E390()              # 计算移动范围
├── sub_18B84()              # 移动路径计算
├── sub_115B6()              # A*寻路
├── sub_4E4F6()              # 路径绘制
├── sub_18D8C()              # 执行移动
│   ├── sub_177FC()          # 循环检查路径
│   ├── sub_1CFF0()          # 移动动画
│   └── sub_1BBDC()          # 攻击动画
└── 更新单位位置+状态
```
